# Jersey Club CR - Frontend

Aplicación frontend profesional para tienda online de camisetas de fútbol.

## Tecnologías
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Zustand
- Framer Motion

## Setup

```bash
cd apps/frontend
npm install
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`
